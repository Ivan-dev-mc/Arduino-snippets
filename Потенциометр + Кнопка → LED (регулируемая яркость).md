## Схема подключения
- **Потенциометр:** левый контакт → 5V, центральный (средний) → A0, правый → GND
- **Кнопка:** один контакт → 5V через резистор 10кΩ (pull-down), второй контакт → пин 2 и через тот же резистор к GND
- **LED:** пин 11 → Резистор 220Ω → Анод (+) → Катод (-) → GND

## Зависимости
- Нет (чистый Arduino C++)

## Код
```cpp
#include <Arduino.h>

// Назначаем имена пинам (читаемость > магия чисел)
#define POT_PIN A0      // Потенциометр на аналоговом входе A0
#define LED_PIN 11      // LED на пине 11 (должен быть PWM-пин!)
#define BUTTON_PIN 2    // Кнопка на пине 2

// Прописываем режим работы пинов (вход/выход)
void setup() {
  pinMode(BUTTON_PIN, INPUT);   // Кнопка — вход (с внешней подтяжкой 10кΩ к GND)
  pinMode(LED_PIN, OUTPUT);     // LED — выход
  Serial.begin(9600);           // Для отладки (смотреть значения)
}

// Основной цикл работы
void loop() {
  // 1. Читаем состояние кнопки
  int buttonstate = digitalRead(BUTTON_PIN);
  
  // 2. Если кнопка нажата (HIGH = 5V через pull-up к 5V)
  if (buttonstate == HIGH) {
    // Читаем значение потенциометра (0-1023, 10 бит ADC)
    int potValue = analogRead(POT_PIN);
    
    // Делим на 4, чтобы перевести в диапазон PWM (0-255)
    // 1023 / 4 = 255.75 → округлится до 255
    int brightness = potValue / 4;
    
    // Подаём PWM-сигнал на LED (регулировка яркости)
    analogWrite(LED_PIN, brightness);
  }
  // 3. Если кнопка НЕ нажата — гасим LED
  else {
    analogWrite(LED_PIN, 0);  // 0 = полностью выключен
  }
  
  // Небольшая задержка для стабильности (10мс)
  delay(10);
}
```

## Что делает
- **Кнопка = "вкл/выкл устройства"**: пока не нажата — LED не горит
- **Потенциометр = регулятор яркости**: крутишь → яркость меняется от 0 до максимума
- **Алгоритм**: читаем кнопку → если нажата, читаем потенциометр → подаём PWM

## Логика работы
1. **`analogRead(POT_PIN)`** возвращает 0-1023 (10 бит ADC Arduino UNO)
2. **`analogWrite(LED_PIN, brightness)`** принимает 0-255 (8 бит PWM)
3. **Деление на 4** — простой маппинг: `1023 → 255`, `0 → 0`
4. **Кнопка с pull-down** (10кΩ к GND): без нажатия = LOW (0V), при нажатии = HIGH (5V)

## Нюансы
- **Пин 11 — PWM-пин** (отмечен тильдой ~ на плате). Если воткнуть в обычный цифровой — LED будет либо гореть, либо нет (без регулировки).
- **Pull-down vs Pull-up**: у тебя 10кΩ к GND → кнопка даёт HIGH при нажатии. Если бы был `INPUT_PULLUP` — логика инвертировалась бы (LOW при нажатии).
- **`delay(10)`** — не обязателен, но стабилизирует чтение. Без него LED может слегка мерцать на крайних положениях потенциометра.
- **Для отладки** добавь в `loop()`:
  ```cpp
  Serial.print("Button: ");
  Serial.print(buttonstate);
  Serial.print(" | Pot: ");
  Serial.print(potValue);
  Serial.print(" | Brightness: ");
  Serial.println(brightness);
  ```
  Увидишь в Serial Monitor все значения в реальном времени.
