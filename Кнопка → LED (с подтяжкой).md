# Схема подключения
- Кнопка: один контакт → GND, второй → пин 2
- LED: пин 13 → Резистор 220Ω → Анод (+) → Катод (-) → GND

## Зависимости
- Нет

## Код
```cpp
const int BUTTON = 2;   // Кнопка на пине 2
const int LED = 13;     // LED на пине 13

void setup() {
  pinMode(BUTTON, INPUT_PULLUP);  // Вход с внутренней подтяжкой к 5V
  pinMode(LED, OUTPUT);
  digitalWrite(LED, LOW);         // Старт: LED выключен
}

void loop() {
  // INPUT_PULLUP: без нажатия пин читает HIGH (5V)
  // При нажатии кнопка замыкает пин на GND → читаем LOW
  if (digitalRead(BUTTON) == LOW) {
    digitalWrite(LED, HIGH);  // Нажали — включили
  } else {
    digitalWrite(LED, LOW);   // Отпустили — выключили
  }
}
```

## Логика
- `INPUT_PULLUP` — внутренняя подтяжка к 5V через резистор ~20кΩ. Кнопка не нужна внешняя.
- Без нажатия: пин = HIGH (ток не течёт, кнопка разомкнута).
- С нажатием: пин = LOW (кнопка замкнула на GND, ток течёт через подтяжку).
- Поэтому проверка = LOW` = «кнопка нажата».

## Нюансы
- Если использовать просто `INPUT` без подтяжки — пин будет ловить помехи и показывать случайные HIGH/LOW.
- Для «toggle» (нажал-отпустил, LED остался включён) нужна переменная состояния + антидребезг (см. отдельный сниппет).
