## Схема подключения
- Пин 13 → Резистор 220Ω → Анод (+, длинная нога) LED → Катод (-, короткая нога) → GND

## Зависимости
- Нет

## Код
```cpp
void setup() {
  pinMode(13, OUTPUT);       // Настраиваем пин 13 как выход
  digitalWrite(13, HIGH);    // Подаём 5V — LED горит
}

void loop() {
  // Пусто. LED горит постоянно, МК не тратит ресурсы
}
```

## Что делает
В `setup()` один раз включаем пин. `loop()` пустой — МК просто крутится вхолостую, ток не меняется.

## Нюансы
- Если подключить к пину без `pinMode(OUTPUT)` — LED будет светить тускло (пин в высоком импедансе).
- `HIGH` = 5V на Arduino UNO, 3.3V на ESP32.
- Резистор 220Ω обязателен: без него LED сгорит за секунды.
