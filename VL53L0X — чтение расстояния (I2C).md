## Схема подключения
- **VL53L0X → Arduino UNO:**
  - VIN → 5V (или 3.3V, если датчик 3.3V-толерантный)
  - GND → GND
  - SDA → A4 (SDA на Arduino UNO)
  - SCL → A5 (SCL на Arduino UNO)
- **Подтяжки:** 10кΩ на SDA и SCL к 5V (или 4.7кΩ, если датчик 3.3V)

## Зависимости
- `Wire.h` (встроенная)
- `VL53L0X.h` (библиотека Pololu) — установить через Library Manager

## Код
```cpp
#include <Arduino.h>
#include <Wire.h>
#include <VL53L0X.h>

VL53L0X sensor;

void setup() {
    Serial.begin(9600);
    delay(2000);  // Ждём открытия Serial Monitor
    
    Serial.println("=== VL53L0X TEST ===");
    Serial.println("Инициализация Wire...");
    Wire.begin();
    
    Serial.println("Проверка датчика...");
    sensor.setTimeout(500);  // Таймаут 500мс (если не ответит — ошибка)
    
    if (!sensor.init()) {
        Serial.println("VL53L0X не найден! Проверь:");
        Serial.println("  SDA -> A4");
        Serial.println("  SCL -> A5");
        Serial.println("  VIN -> 5V");
        Serial.println("  GND -> GND");
        Serial.println("  Pull-up 10кОм на SDA и SCL к 5V");
        while (1);  // Зависаем, пока не исправим
    }
    
    Serial.println("Датчик найден! Запуск измерений...");
    
    // Timing budget: время на одно измерение
    // 200000 мкс = 200мс — надёжно, но медленно
    // 33000 мкс = 33мс — быстро, но менее точно
    sensor.setMeasurementTimingBudget(200000);
        // Continuous mode: датчик меряет каждые 100мс
    sensor.startContinuous(100);
}

void loop() {
    // Читаем расстояние в миллиметрах
    uint16_t distance = sensor.readRangeContinuousMillimeters();
    
    // Проверка на timeout (датчик не успел измерить)
    if (sensor.timeoutOccurred()) {
        Serial.println("TIMEOUT (65535)");
    } else {
        Serial.print("Расстояние: ");
        Serial.print(distance);
        Serial.println(" мм");
    }
    
    delay(100);  // Пауза между чтениями
}
```

## Что делает
- Инициализирует I2C шину (`Wire.begin()`)
- Проверяет, отвечает ли датчик на адрес 0x29
- Если не отвечает — выводит чек-лист и зависает
- Если отвечает — настраивает timing budget (200мс) и запускает continuous mode
- В цикле читает расстояние каждые 100мс
- Если timeout — выводит ошибку, иначе — расстояние в мм

## Логика работы
1. **`Wire.begin()`** — инициализирует Arduino как I2C master
2. **`sensor.setTimeout(500)`** — если датчик не ответит за 500мс → timeout
3. **`sensor.init()`** — ищет датчик на шине, инициализирует его
4. **`sensor.setMeasurementTimingBudget(200000)`** — время на одно измерение:
   - 200мс = высокая точность, но медленно
   - 33мс = быстро, но шумнее
   - Для Runout_Scanner: 200мс — ок (не нужна высокая скорость)
5. **`sensor.startContinuous(100)`** — датчик меряет каждые 100мс в фоне
6. **`sensor.readRangeContinuousMillimeters()`** — читает последнее измерение
7. **`sensor.timeoutOccurred()`** — проверяет, был ли timeout (возвращает 65535)

## Нюансы
- **Подтяжки 10кΩ** — работают, но стандарт I2C рекомендует 4.7кΩ. Если датчик 3.3V — обязательно 4.7кΩ к 3.3V
- **Timing budget** — компромисс между скоростью и точностью:
  - 33мс → ~30 измерений/сек (быстро, но шум)
  - 200мс → ~5 измерений/сек (медленно, но точно)
  - Для Runout_Scanner: 200мс — ок, диск вращается медленно
- **Continuous mode** — датчик меряет в фоне, ты просто читаешь результат. Не блокирует МК
- **Timeout** — если датчик не успел (например, объект слишком далеко/близко) → возвращает 65535
- **Диапазон:** 30мм — 2000мм (3см — 2м). Ближе 30мм — не меряет- **Точность:** ±3% (на 100мм → погрешность ±3мм)

## Типичные ошибки
| Ошибка | Симптом | Решение |
|--------|---------|---------|
| Нет подтяжек | "VL53L0X не найден" | Добавь 4.7-10кΩ на SDA/SCL к 5V |
| Неверный адрес | "VL53L0X не найден" | Запусти I2C Scanner, проверь адрес (обычно 0x29) |
| Перепутал SDA/SCL | "VL53L0X не найден" | SDA → A4, SCL → A5 (для Arduino UNO) |
| Питание 5V на 3.3V датчике | Датчик сгорел | Проверь даташит! VL53L0X = 2.6-3.3V |
| Объект ближе 30мм | Возвращает 65535 или мусор | Датчик не меряет ближе 30мм |
| Объект дальше 2м | Возвращает 65535 | Датчик не меряет дальше 2м |

## Формула для Runout_Scanner

```
baseline = среднее(50 замеров)
sigma = стандартное_отклонение(50 замеров)

если |замер - baseline| < 2σ → Зелёный (норма)
если |замер - baseline| < 4σ → Жёлтый (внимание)
если |замер - baseline| >= 4σ → Красный (критично) 
```
