## Схема подключения
- **Потенциометр:** левый → 5V, центральный → A0, правый → GND
- **Кнопка:** один контакт → 5V, второй → пин 2 + резистор 10кΩ к GND (pull-down)
- **LED:** пин 11 → резистор 220Ω → анод (+) → катод (-) → GND

## Зависимости
- Нет

## Код
```cpp
#include <Arduino.h>

#define POT_PIN A0
#define LED_PIN 11
#define BUTTON_PIN 2

bool isPowerOn = false;        // Виртуальный выключатель (toggle)
int lastButtonState = HIGH;    // Предыдущее состояние кнопки

void setup() {
  pinMode(BUTTON_PIN, INPUT);      // Кнопка с внешней подтяжкой к GND
  pinMode(LED_PIN, OUTPUT);
  Serial.begin(9600);
}

void loop() {
  int buttonState = digitalRead(BUTTON_PIN);
  
  // 1. Детекция фронта: ловим момент изменения состояния
  if (buttonState != lastButtonState) {
    
    // 2. Реагируем только на нажатие (HIGH), не на отпускание
    if (buttonState == HIGH) {
      isPowerOn = !isPowerOn;  // Инвертируем флаг
      Serial.println(isPowerOn ? "СИСТЕМА ВКЛ" : "СИСТЕМА ВЫКЛ");
    }
    
    delay(50);  // Антидребезг: ждём успокоения контактов
  }
  
  // 3. Запоминаем состояние для следующего круга
  lastButtonState = buttonState;
  
  // 4. Основная логика работает по флагу, а не по кнопке
  if (isPowerOn) {
    int potValue = analogRead(POT_PIN);  // 0-1023
    analogWrite(LED_PIN, potValue / 4);  // 0-255 (PWM)
  } else {
    analogWrite(LED_PIN, 0);  // Выключено
  }
}
```

## Что делает
- **Кнопка = toggle**: нажал → включилось, нажал ещё → выключилось
- **Потенциометр = регулятор**: работает только когда система включена
- **Логика разделена**: кнопка управляет флагом, флаг управляет LED

## Ключевые концепции
- **Edge detection** — реакция на изменение, а не на состояние
- **Toggle** — инвертирование флага по событию
- **Разделение ответственности** — ввод (кнопка) ≠ обработка (флаг) ≠ вывод (LED)
- **Антидребезг** — `delay(50)` для стабилизации механических контактов

## Нюансы
- `delay(50)` блокирует МК. Для сложных проектов — заменить на `millis()` (см. Глава 1.4)
- По конвенции C++ переменные пишутся camelCase: `lastButtonState`, не `LastButtonState`
- Если нужна реакция на **удержание** (долгое нажатие) — добавляй счётчик времени в `loop()`
